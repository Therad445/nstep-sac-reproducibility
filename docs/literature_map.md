# Literature Map

## SAC

Soft Actor-Critic is an off-policy actor-critic algorithm for continuous control. It combines a stochastic policy, twin Q-critics and entropy regularization. The entropy term encourages exploration and makes the policy less brittle during training.

In the standard version, the critic is usually trained with a one-step TD target. This is stable and simple, but the reward signal can propagate slowly in long-horizon tasks.

## n-step returns

N-step returns use several rewards before bootstrapping from the critic. In an on-policy setting this often improves credit assignment. In an off-policy setting it is more delicate: the trajectory in the replay buffer was generated by an older policy, so the multi-step target can be biased.

The main trade-off is simple:

- larger `n` can propagate rewards faster;
- larger `n` can also increase variance and off-policy mismatch.

## SACn

SACn studies how to add n-step returns to SAC without making the algorithm unstable. The important point for this project is not only the proposed method, but the motivation: naive n-step SAC is not a free improvement. It needs a careful discussion of bias, variance and stability.

## T-SAC

T-SAC moves the idea further. Instead of giving the critic only a single state-action pair, it conditions the critic on short trajectory chunks and uses a lightweight Transformer. This is meant to improve credit assignment and temporal context handling.

## Place of this project

This project does not reproduce the full T-SAC training setup. It takes the central mechanism, n-step credit assignment, and tests it in a smaller setting. The result is a bounded reproduction and ablation study rather than a new algorithm paper.
